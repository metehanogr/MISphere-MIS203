About MISphere

MISphere is a classic-style web project designed for MIS203. It shows how a simple multi-page website for MIS students can be organised using HTML, CSS and a small amount of JavaScript.

Purpose of the Website

The main aim of MISphere is to provide a clean, text-based hub for Management Information Systems students. The site focuses on structure and readability rather than heavy graphics.

  • Collect course-related announcements in one place.
  • List upcoming events such as guest lectures and workshops.
  • Offer a simple forum-style page for discussion topics.
  • Show a basic example of a student profile page.

Who is MISphere for?

MISphere is mainly intended for undergraduate MIS students who want to see a concrete example of a small course website.

  • Students taking MIS203 – Web Technologies for MIS.
  • Students interested in decision-support and web systems.
  • Project groups preparing their own term project websites.

How it is structured

The website consists of several static pages sharing the same layout. This helps demonstrate a consistent design across the project.

  • Common navigation bar on all pages.
  • One main stylesheet file: assets/style.css.
  • A small script file, assets/scripts.js, for demo login and footer year.
